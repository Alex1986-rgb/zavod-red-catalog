# Завод Редукторов — демо редизайна

Демонстрационный макет нового дизайна сайта **zavod-red.ru** (интернет-магазин промышленных редукторов и мотор-редукторов).

**Живая демонстрация:** https://alex1986-rgb.github.io/zavod-red-catalog/

Открывается по ссылке с любого компьютера, без установки.

## Страницы

| Страница | Файл |
|---|---|
| Главная | `index.html` |
| Каталог с фильтрами | `catalog.html` |
| Карточка товара | `product.html` |
| Подбор по параметрам | `podbor.html` |
| Калькулятор стоимости | `calculator.html` |
| Корзина | `cart.html` |
| Оформление заказа | `checkout.html` |
| Личный кабинет клиента | `cabinet.html` |
| О заводе | `about.html` |
| Блог / статья | `blog.html` / `article.html` |

## Что работает в демо

- **Светлая тема по умолчанию** + переключатель светлая/тёмная (выбор сохраняется между страницами).
- **Корзина**: добавление товара, счётчик в шапке, изменение количества (сохраняется в браузере).
- **FAQ-аккордеон**, вкладки, слайдеры подбора, фильтры-чипы.
- Навигация между всеми страницами.
- SEO: заголовки, описания, Open Graph, адаптивная вёрстка.

Статический сайт (HTML/CSS/JS без сборки, изображения встроены). Публикуется через GitHub Pages из ветки `main` — workflow `.github/workflows`.
