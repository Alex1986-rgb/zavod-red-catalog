# Завод Редукторов — сайт-каталог

Статический многостраничный сайт для публикации через GitHub Pages.

## Страницы

| Файл | Страница |
|------|----------|
| `index.html` | Главная |
| `catalog.html` | Каталог редукторов |
| `about.html` | О заводе |
| `contacts.html` | Контакты |
| `blog.html` | Блог |

Все кнопки меню и ссылки ведут на реальные страницы — переходы работают.

## Публикация на GitHub Pages

1. Запушьте изменения в ветку `main`.
2. GitHub → Settings → Pages → Deploy from a branch → `main` → `/root` → Save.
   (В репозитории уже есть workflow `.github/workflows/pages.yml`, который делает это автоматически при пуше в `main`.)
3. Сайт откроется по адресу `https://<username>.github.io/zavod-red-catalog/`.

## Технические заметки

- Дизайн собран из макета (вкладка «Дизайн»). Страницы — самодостаточный HTML с инлайн-стилями,
  шрифты (Archivo, Space Grotesk, IBM Plex Mono) подключаются с Google Fonts.
- Изображения товаров грузятся с внешнего домена `zavod-red.ru` — нужен интернет.
- Интерактив каталога (живые фильтры, корзина, переключение темы, поиск) в статической версии
  **не активен** — это визуальный каркас. Переходы между страницами работают полностью.
- Телефон в шапке (`+7 904 953-41-02`) и в подвале/контактах (`+7 495 151-41-02`) различаются —
  унаследовано из макета, при необходимости приведите к одному номеру.
